# 安装与设置
## 基于 openSUSE 和 SLE 的发行版安装 Visual Studio Code
1. 使用以下 rpm 命令导入 Microsoft 的 GPG 密钥：

```bash
sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
```

2. 使用 Microsoft 的 GPG 密钥，继续并为 Visual Studio Code 创建存储库文件：

```bash
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ntype=rpm-md\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/zypp/repos.d/vscode.repo'
```

3. 更新包缓存并使用以下命令安装 Visual Studio Code：

```bash
sudo zypper refresh
sudo zypper install code
```

## 安装中文输入法
OpenSUSE下可以使用`中州韵`输入法，该输入法运行需要`fcitx5`的支持：
1. 更新系统包：

```bash
sudo zypper refresh
```

2. 安装`fcitx5`框架（如果尚未安装的话）：

```bash
sudo zypper install fcitx5
```

3. 安装`fcitx5-rime`输入法（也称作`中州韵`输入法）：

```bash
sudo zypper install fcitx5-rime
```

4. 配置`fcitx5`为默认输入法：

打开系统设置。
进入<font face="微软雅黑" color=yellow>**区域与语言**</font>或类似选项。
在输入法选项中，选择fcitx5作为默认输入法。

5. 重新启动计算机或重新登录以应用修改。

## 更换国内软件源

1. 首先禁用官方软件源：

```bash
sudo zypper mr -da
```

2. 添加清华镜像源：

```bash
sudo zypper ar -cfg 'mirrors.tuna.tsinghua.edu.cn/opensuse/tumbleweed/repo/oss/' mirror-oss
sudo zypper ar -cfg 'https://mirrors.tuna.tsinghua.edu.cn/opensuse/tumbleweed/repo/non-oss/' mirror-non-oss
```

3. 刷新软件源

```bash
sudo zypper refresh
```

> 提示：
> 对于 Tumbleweed，Tumbleweed 安装后默认会启用 oss, non-oss, update, 3 个官方软件源，其中 oss 及 non-oss 源用于发布 Tumbleweed 的每日构建快照，快照中已包含系统所需的全部软件包及更新。update 源仅用于推送临时安全补丁，如当日快照已发布但仍有临时安全补丁时，会首先推送至 update 源，并在次日合入下一版快照。 由于 update 源存在较强的时效性，上游镜像并未同步 update 源， LZUOSS 亦无法提供该源的镜像。禁用 update 源并不会使系统缺失任何功能或安全更新，仅会导致极少数更新晚些推送，如有需求可以重新启用官方 update 源。

## 安装Docky
OpenSUSE默认源里不支持`cairo-dock`，那就只能安装`plank`了：

```bash
sudo zypper install plank
```
安装完成后记得在<font face="微软雅黑" color=yellow>**设置管理器**</font>中的<font face="微软雅黑" color=yellow>**会话和启动**</font>配置一下将`plank`设置为自启动模式。
