# 5.10版本TF-a调试

烧写tf-a和fip.bin步骤：

（1）去除eMMC的boot0和boot1分区写保护：

```bash
echo 0 > /sys/block/mmcblk1boot0/force_ro 
echo 0 > /sys/block/mmcblk1boot1/force_ro
```

（2）用dd命令烧写boot0和boot1分区：

```bash
dd if=tf-a-stm32mp157d-ev1.stm32 of=/dev/mmcblk1boot0
dd if=tf-a-stm32mp157d-ev1.stm32 of=/dev/mmcblk1boot1
```

（3）用dd命令烧写U-boot分区：

```bash
dd if=fip.bin of=/dev/mmcblk1p1
```

（4）将原来4.19下的U-boot分区的名称改为“fip”：

```bash
sgdisk -c 1:"fip" /dev/mmcblk1
```

（5）sync并重启。
