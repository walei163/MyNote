# 无线以太网的调试过程

利用wpa_supplicant命令连接wifi
wpa参数配置：

vi /etc/wpa_supplicant.conf

添加/修改如下内容：

        ctrl_interface=/var/run/wpa_supplicant
        #ap_scan=1
         
        network={
            ssid="name"
            psk="passwd"
        }
ssid对应wifi名，psk对应密码

连接wifi

        wpa_supplicant -Dnl80211 -iwlan0 -c/etc/wpa_supplicant.conf &

ifconfig查看状态

        wlan0     Link encap:Ethernet  HWaddr FC:3D:93:B4:E9:9D
