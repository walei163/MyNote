# DCA-1600产品相关

`DCA-1600`产品在克潇项目上使用时，需要修改文件系统。

其文件系统的目录：[文件系统](/media/jason/develop/克潇产品项目移植运行时库arm-linux-gnueabihf-20230428/最新文件系统_20230615/)

升级文件可以采用`nfs`文件系统模式来将装置上的文件系统进行替换，方法如下：

1、可以修改`extlinux.conf`文件，增加以下内容:

```sh
LABEL sanway_cce3088_nfs
        KERNEL /uImage_5.10
        FDT /sanway_cce3088-eb-a.dtb
        APPEND console=ttySTM0,115200n8 root=/dev/nfs nfsroot=192.168.1.177:/media/ubuntu16.04/mnt/sanway-emmc-rootfs-4.9-stm32mp153a-swa1530,nolock,vers=3 rw ip=192.168.1.223:192.168.1.177:192.168.1.1:255.255.255.0::eth0:off
```

2、启动装置，选择从`nfs`启动，然后进入到系统根目录下：

```bash
mount /dev/mmcblk1p4 /mnt
rm -rf /mnt/*
tar xzvf rootfs_20240506_KeXiao_dca1600.tar.gz -C /mnt
sync
```

3、重启系统，会加载新的文件系统，然后还需要修改`/etc/init.d/swa3300.sh`，将：

```sh
#mount -t ext4 /dev/mmcblk1p5 /usr/local
```

注释去掉，以便/usr/local/下次启动可以正常挂载。

>[!note]
>
>1、`DCA-1600`采用的是`STM32MP153D`处理器。
>
>2、注意引导内核版本为`5.10.10`。
