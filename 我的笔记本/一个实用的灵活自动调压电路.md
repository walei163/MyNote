# 一个实用的灵活自动调压电路

2025-05-26 18:20·[朽木逢春](https://www.toutiao.com/c/user/token/MS4wLjABAAAAEqZUYofNQNR4Ok_EutuDqdakQn9scQyEkKZuw5bqMSI/?source=tuwen_detail)

前段时间在一个产品测试中需要一个3.5V，3.7V和4.2V可控电压的供电电源，网上找了一圈也没有符合要求的模块。终究还是得自己做一个，既然要做，那就做个灵活可控的可调电压源。

![img](https://p3-sign.toutiaoimg.com/tos-cn-i-axegupay5k/78551790b3ea4a2e89db022f9043d1ea~tplv-tt-origin-web:gif.jpeg?_iz=58558&from=article.pc_detail&lk3s=953192f4&x-expires=1753862601&x-signature=LJNb7vuvxoSMeiuaAcJYRWp5aHk%3D)



调压电路并不复杂，主要元器件是MIC29302和TPL0501。

MIC29302是一款输出电压可调的稳压芯片，可以通过改变5脚上的电阻R1和R2阻值来调节输出电压值。

![img](https://p3-sign.toutiaoimg.com/tos-cn-i-6w9my0ksvp/d830bdbcd2cf4dd1a71c1ed6c4fb093e~tplv-tt-origin-web:gif.jpeg?_iz=58558&from=article.pc_detail&lk3s=953192f4&x-expires=1753862601&x-signature=O76sO%2FjaEe%2BPxquyCt6ZVTrf%2BNE%3D)



根据公式Vout = 1.24*(1+R1/R2)，在电路设计时，R1采用固定阻值电阻，R2采用可调电阻，这样通过调节R2阻值来改变输出电压值，为了能够灵活自动输出所需电压，可调电阻我们选用数字电位器。

TPL0501就是是一个具有SPI接口的数字电位器。具有256个游标位置，最大阻值100kΩ，最小单位阻值可以到390Ω，可以通过SPI接口来控制游标的位置，配置所需阻值。

在实际应用中为了保护电路，避免可调电阻阻值为0，我们在数字电位器上串联一个小阻值的电阻。可调电源的电路原理图如下所示，感兴趣的可以了解下，有几个注意的地方在此说明一下。

![img](https://p3-sign.toutiaoimg.com/tos-cn-i-6w9my0ksvp/46ef57c64af24e0f919a9b386936ed29~tplv-tt-origin-web:gif.jpeg?_iz=58558&from=article.pc_detail&lk3s=953192f4&x-expires=1753862601&x-signature=%2FNbpuzaRnOQyRMNuQyF6309e3bs%3D)



1. 输入Ctrl通过一个Q1 NMOS管使能MIC29302。
2. 当输入Vin电压为6V时，数字电位器配置的阻值范围为23~100kΩ，Vout可以输出电压范围在2.4V~5V。
3. 可以用MCU的ADC实时监测Vout的电压值，动态的调节数字电位器游标位置，保证输出恒压。
4. 可以根据工作需求改变Vin的输入电压，修改R1，R2的阻值，选择不同参数的数字电位器，达到不同的电压输出范围。

电路也许有不足之处，也还有很多可以改进的地方，欢迎大家讨论。
